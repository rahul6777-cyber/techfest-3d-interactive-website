# 🌐 Techfest 3D Interactive Website

A modern 3D interactive website developed for the Techfest Web Development Challenge. The project focuses on immersive visuals, smooth animations, interactive 3D elements, and a responsive user experience.

---

## 🚀 Features

- 🌍 Interactive 3D Environment
- ✨ Smooth Scroll Animations
- 🎮 Interactive 3D Objects
- 🎨 Modern UI/UX Design
- ⚡ Dynamic Transitions
- 📱 Fully Responsive Design
- 🌌 Animated Background
- 🖱️ Mouse Interaction
- 🚀 Optimized Performance

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Three.js
- GSAP (if used)

## ▶️ Getting Started

1. Clone the repository

```bash
git clone https://github.com/rahul6777-cyber/techfest-3d-interactive-website.git
```

2. Open the project folder.

3. Run the project using **Live Server** or open `index.html` in your browser.
